The Importance of Code Quality

High code quality leads to more maintainable, reliable, and functional code. Without considering code quality you will spend a significantly greater amount of time attempting to fix bugs, improve performance, collaborate effectively, detect vulnerabilites, and etc. This platform can provide you and your team with a starting point for improving the quality of your code.

Available Metrics

Cyclomatic Complexity:
Counts number of decision points in a code base

Code Documentation:
Checks percentage of code that is documented

Code Coverage:
Checks percentage of code covered by automated testing

Code Style:
Ensures code adheres to established practices and standards

Code Duplication:
Identifies percentage of duplicated code
